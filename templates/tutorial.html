Tutorial

AlgoJ is a python based judge system that you can use for testing your source code. The archive contains a list of algorithmic tasks which you must solve.

Registration

To access the archive, you first need to register. The registration is a simple process, you can use your university email or your personal one. Press the Register button in the top right corner in order to register.

Problem Archive

The problem archive contains a list of tasks available for solving. Each task is composed of a:

  1. Problem statement
  2. Implementation specification
  3. One or more examples

Submission

In order to solve a problem, you must write a function(s) that conforms to the specification of the problem.
